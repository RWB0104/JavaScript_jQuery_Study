node.js 오류

node.js는 자바스크립트를 프로그래밍 언어로 사용하므로 방금 살펴본 오류 모두 발생합니다. 그리고 지금부터 알아볼 오류가 추가로 발생합니다.

Cannot find module "경로"

처음 node.js 애플리케이션을 공부할 때 자주 보는 오류입니다. 애플리케이션 파일이 현재 폴더에 없을 경우 뜹니다.

Cannot find module "식별자"

Connot find module "express"처럼 모듈이 없다고 하는 경우입니다. 모듈이 설치되지 않았을 때 발생하는 오류입니다. "npm install 식별자" 명령어로 모듈을 설치하세요.

listen EADDRINUSE

다른 프로그램이 같은 포트를 사용하고 있을 경우 발생하는 오류입니다. 다음과 같은 두 가지 경우가 원인이 됩니다.


1. 다른 프로그램이 같은 포트를 사용하고 있을 경우

2. 같은 프로그램을 두 번 실행했을 경우


책에서는 52273 포트를 사용하는데, 일반적으로 사용되지 않는 포트입니다. 따라서 1번 경우 보다는 2번 경우가 많습니다. 다른 명령 프롬프트로 프로그램을 실행하고 있지 않은지 살펴보세요!
