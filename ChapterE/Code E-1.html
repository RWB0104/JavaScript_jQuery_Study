자바스크립트 오류

자바스크립트라는 프로그래밍 언어 자체에서 발생할 수 있는 오류를 알아보겠습니다.

Unexpected Identifier.

문장 구조 자체가 틀렸을 때 발생하는 오류입니다. 문장을 잘못 입력하지 않았는지 살펴보세요.

"식별자" is not defined.

존재하지 않는 식별자를 사용할 때 발생하는 오류입니다. 식별자를 선언했는지 확인하세요.

Cannot call method "식별자" of undefined

다음과 같은 경우에 발생하는 오류입니다.

<script>

  test.method();

</script>

존재하지 않는 객체의 메서드를 호출하려는 경우입니다. 이러한 경우에는 "식별자(예에서는 test)"를 잘못 입력하지 않았는지 살펴보세요.

Object <자료형> has no method "식별자"

다음과 같은 경우에 발생하는 오류입니다.

<script>

  var test = { };

  test.method();

</script>

Cannot call method "식별자" of undefined와 다르게 객체는 존재하지만 메서드가 없는 경우입니다. 이러한 경우에는 "식별자(예에서는 method)"를 잘못 입력하지 않았는지 살펴보세요.
